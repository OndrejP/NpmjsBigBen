BigBen
===========

Bong every 15 minutes to npmjs.org.
Create new version of this project to check functionality of npmjs.org.

Version style: 

``` 
used: YYYY.DDD.HHMM
cmd: date +"%Y.%j.%H%M"
example: 2014.258.1258 
example realtime: Mon Sep 15 12:58:00 CEST 2014
```

###FAQ:
